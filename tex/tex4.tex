\section{現在稼動中のWebアプリのデータベースをMySQLに置き換える}

第13章の employeesテーブルを使ったWebアプリケーションを
H2データベース から MySQLに置き換える。\\

\textgt{必要な手順}

\begin{enumerate}
 \item mysql-connector-j-8.2.0.jar を src/main/webapp/WEB-INF/lib に
       配置する。
 \item EmployeesDAO.java の JDBC\_URL を書き換える。
 \item EmployeesDAO.java の Class.forName() を書き換える。
\end{enumerate}


\begin{lstlisting}[title=(参考1) EmployeesDAO.java (部分)]
private final String JDBC_URL =
		"jdbc:mysql://localhost:3306/example";
\end{lstlisting}


\begin{lstlisting}[title=(参考2) EmployeesDAO.java (部分)]
try {
  Class.forName("com.mysql.cj.jdbc.Driver");
} catch (ClassNotFoundException e) {
  throw new IllegalStateException
    ("JDBCドライバの読み込みエラー");
}		

\end{lstlisting}

これだけでよい。
